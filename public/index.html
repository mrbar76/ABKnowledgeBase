How to Export Your Data

ChatGPT:
chatgpt.com → Profile → Settings → Data Controls → Export Data
Unzip the download and upload conversations.json

Claude:
claude.ai → Settings → Export Data
Upload the conversations file from the export

Bee Wearable Sync Script

Syncs facts, todos, and conversations from your Bee wearable into AB Brain.

# Install prerequisites npm install -g @beeai/cli bee login # Run sync (from repo root) ./scripts/bee-to-brain-sync.sh # Sync only specific data ./scripts/bee-to-brain-sync.sh --only facts ./scripts/bee-to-brain-sync.sh --only todos ./scripts/bee-to-brain-sync.sh --only convos # Sync last 7 days of conversations ./scripts/bee-to-brain-sync.sh --recent-days 7 # Schedule daily at 8am (add to crontab -e) 0 8 * * * /path/to/scripts/bee-to-brain-sync.sh >> ~/bee-sync.log 2>&1
